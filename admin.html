글 관리

이 페이지에서 글을 쓰고 "발행"을 누르면 GitHub API를 통해 저장소의 posts.json이 갱신되고, 잠시 후(보통 1분 내) 블로그에 자동으로 반영됩니다.

1. 인증

토큰은 이 브라우저 탭의 메모리에만 잠깐 사용되고 어디에도 저장되지 않습니다. 탭을 닫으면 사라지므로 매번 붙여넣어야 해요.
발급: GitHub → Settings → Developer settings → Fine-grained tokens → Repository access를 이 저장소 하나만 선택 → Permissions에서 Contents: Read and write만 부여.

알아두기
  • 이 페이지(admin.html)는 공개 저장소에 있어서 누구나 열어볼 수 있지만, 토큰 없이는 아무것도 발행할 수 없습니다.
  • 발행할 때마다 저장소에 커밋이 하나씩 생깁니다 (커밋 내역 = 글 수정 이력).
  • 발행 후 사이트 반영까지 GitHub Pages 빌드 시간(수십 초~1분)이 걸립니다.
